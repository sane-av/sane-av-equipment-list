# SANE Equipment Database

Open community-maintained database of professional AV equipment specifications. All data is CC BY-SA 4.0.

## Structure

Equipment data is stored as JSON files organized by category (`amplifiers.json`, `displays.json`, etc.). A `schema.json` defines the expected fields for each entry.

## Submitting Data

Open an issue using the equipment submission template. Include manufacturer, model, category, and specifications.
